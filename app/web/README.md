# MMFOOD Web Frontend

Next.js 14 web application for multilingual recipe search.

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Features

- Text and voice search
- Multilingual support (English, Hindi, Marathi, Tamil, etc.)
- Advanced filters (cuisine, diet, time, exclusions)
- Mobile-first responsive design
- Recipe details modal

## Technology

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Hook Form + Zod
- Axios for API calls
